Dynamic Pelican settings

By Stefaan Lippens on 2016/10/06
Tagged: pelican 
class DynamicSetting(object):
    """
    Allow a Pelican setting, such as ARTICLE_URL and ARTICLE_SAVE_AS, to be dynamic.
    """
    def __init__(self, f):
        self._f = f
    def format(self, **kwargs):
        return self._f(kwargs).format(**kwargs)

With this helper class you can create dynamic settings like in this example 

ARTICLE_URL = DynamicSetting(lambda metadata: 
    '{cleanurl}' if 'cleanurl' in metadata else '{slug}.html')
ARTICLE_SAVE_AS = DynamicSetting(lambda metadata: 
    '{cleanurl}/index.html' if 'cleanurl' in metadata else '{slug}.html')

You can even, if you prefer that, use it as decorator, e.g.:

@DynamicSetting
def ARTICLE_URL(metadata):
    if 'cleanurl' in metadata:
        return '{cleanurl}'
    else:
        return '{slug}.html'
